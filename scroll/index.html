What is Angular?
Angular is a JS framework that helps build applications that run in a web.

Who developed AngularJS?
Google is the company that developed AngularJS. AngularJS is an open source project.

AngularJS is an excellent framework for building both SPA and MPA projects.

What are the benefits of using AngularJS? Dependency Injection: Something that AngularJS does quite well. If you are new to Dependency Injection, don't worry.

What is ng-app?
In angular, ng-app is called a directive. There are many directives included in angular. The ng prefix in the directive stands for angular. The ng-app directive is a starting point of AngularJS application. Angular framework will first check for ng-app directive in an HTML page, after the entire page is loaded. If ng-app directive is found, angular boostraps itself starts to manage the section of the page that has the ng-app directive.

So, the obvious next question is, where to place the ng-app directive on the page?
It should be placed at the root of the HTML document, that is at the html tag level or at the body level, so that angular can control the entire page.
However, there is nothing stopping you from placing it on any other HTML element within the page. When you do this only, that element and it's children are managed by Angular.
